```javascript
// Example: New collaboration API usage
const collaborate = async (taskId, userId) => {
  const response = await fetch(`/api/tasks/${taskId}/collaborate`, {
    method: 'POST',
    body: JSON.stringify({ userId })
  });
  return response.json();
};
```
</Update>

## Upgrading Loz Turner

Follow these steps to update to the latest version and apply changes.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your current tasks and dashboards.

    ```bash
    loz-turner export --backup tasks.json dashboards.json
    ```
  </Step>
  <Step title="Install Update" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm update loz-turner
        ```
      </Tab>
      <Tab title="Docker" icon="docker">
        ```bash
        docker pull lozturner/app:latest
        docker-compose up -d
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify Changes" icon="check-circle">
    Check the changelog above and test key features like collaboration.
  </Step>
</Steps>
